## Purpose

Reference principle to prioritize visual expression over verbose text for understanding project structure and decisions.

## Source

- principle: `千言万语不如一张图`
- role: guide for graph-first and card-first information delivery

## Intended Scope

1. prefer visual graph/card outputs for asset relationships and state
2. compress long textual explanations into visual summaries when possible
3. surface key status through glanceable visual cues

## Non-Scope

- no decorative visuals without decision value
- no visual complexity that increases cognitive load
